## What is AI-Workshop?

**AI-Workshop** is a mission-based AI education platform designed to take you from AI beginner to practitioner through hands-on, real-world challenges — modeled after the way special operations teams are trained.

## Mission Tracks

| Track | Difficulty | Description |
|-------|-----------|-------------|
| **Recruit** `•` | Foundational | Core AI concepts, API basics, prompt engineering |
| **Operative** `••` | Intermediate | Agents, tool use, RAG, multi-agent systems |
| **Commander** `•••` | Advanced | Production systems, AI architecture (coming soon) |
| **Special Ops** | Standalone | MCP, Claude Code, AI Security — no prerequisites |

## How to Use This Platform

1. Start with **RECRUIT-00: Welcome to AI-Workshop** to get oriented
2. Complete missions in order within each track
3. Special Ops missions can be taken at any time
4. Use the comments section on each mission to ask questions
5. Open an [Issue](https://github.com/akashtalole/AI-Workshop/issues) for feedback or bugs

## Contributing

Contributions are welcome! To add a new mission:

1. Fork the repository
2. Create a post in `_posts/` following the mission template
3. Use the appropriate category (`Recruit`, `Operative`, `Commander`, `Special-Ops`)
4. Open a pull request with your mission
